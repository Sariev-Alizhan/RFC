# RFC WhatsApp продажник 🤖🔴

Бот-продажник для **Red Flag Community**. Подключается к WhatsApp сканированием QR с телефона, отвечает клиентам от вашего номера. Гибрид: сценарии на частые вопросы (каталог, размеры, доставка, оплата, оформление заказа) + Claude AI на свободные вопросы.

---

## ⚠️ Важно перед запуском

Это **неофициальный** способ (WhatsApp Web протокол, библиотека Baileys). Он рабочий, но:

- **Против правил WhatsApp** → есть риск бана номера, особенно при массовых рассылках. Бот тут только **отвечает** на входящие (не спамит) — это снижает риск, но не убирает совсем.
- Рекомендую подключать на **рабочий номер магазина**, а не личный.
- Официальная альтернатива без риска — WhatsApp Business API (платный, через Meta/провайдера). Скажи, если захочешь перейти на неё.

## 🚀 Запуск (локально, на твоём компе)

Нужен **Node.js 18+** ([nodejs.org](https://nodejs.org)).

```bash
cd wa-bot
npm install

# (необязательно) включить AI-ответы:
cp .env.example .env
# открой .env и вставь CLAUDE_API_KEY

npm start
```

1. В терминале появится **QR-код**.
2. На телефоне: WhatsApp → **Настройки → Связанные устройства → Привязка устройства** → отсканируй QR.
3. Готово — увидишь «✅ Подключено!». Бот отвечает на входящие.

> Не закрывай окно терминала — бот работает, пока оно открыто. Сессия сохраняется в папке `auth/`, повторно QR сканировать не нужно.

## 🧠 Как работает

- **Сценарии** (бесплатно, мгновенно): привет, каталог, размеры, доставка, оплата, оформление заказа по шагам (товар → размер → имя → город → подтверждение с суммой и ссылкой Kaspi).
- **AI** (если задан `CLAUDE_API_KEY`): всё остальное — консультирует в тоне RFC, подводит к заказу. Модель по умолчанию — Haiku 4.5 (дёшево).
- **Авто-передача человеку:** если клиент пишет «менеджер»/«человек» — бот замолкает на 30 мин. Если **ты сам** пишешь в этот чат с телефона — бот тоже автоматически замолкает на 30 мин, чтобы не мешать живому диалогу.

Данные магазина (товары, цены, тексты) — в `shop.js`. Меняй там при обновлении.

## 🔧 Чтобы работал без открытого терминала (опционально)

Держать всегда включённым на компе через PM2:

```bash
npm install -g pm2
pm2 start index.js --name rfc-wa
pm2 save
```

Но пока комп выключен/спит — бот офлайн. Для «всегда онлайн» нужен облачный хост (Railway/VPS) — скажи, помогу перенести.
